# External Media Policy

This repository uses lightweight GIFs and still images instead of raw full-length MP4 files.

## Public Clips Included In Repo

| Public title | Repo asset | Why this is enough for GitHub review |
| --- | --- | --- |
| Physical hallway balancing demo | `media/hero/physical_balance_hallway.gif` | Shows the clearest continuous balancing run |
| Physical obstacle-course balancing demo | `media/demos/physical_balance_obstacle_course.gif` | Shows controlled physical movement around simple obstacles |
| Robot-focused hallway still | `media/demos/hallway_robot_only.jpg` | Provides a still image without making the operator the focus |

## Full-Length Originals

Full raw MP4 clips are not committed here. External publication should require trimming, anonymization, and a clear technical reason beyond the existing GIFs.
