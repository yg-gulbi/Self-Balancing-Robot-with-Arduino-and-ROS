# Archive

This folder keeps older project material that still helps explain how the robot was developed, but it is not the best place to start if you are seeing the project for the first time.

If you are new to the repository, start with [`../README.md`](../README.md), [`../firmware/physical_balance_controller`](../firmware/physical_balance_controller/README.md), and [`../ros_ws`](../ros_ws/README.md). Come here when you want to see the older experiments and background material behind those main entrypoints.

## Folders

| Folder | What it contains | When to read it |
| --- | --- | --- |
| [`ros_experiments`](ros_experiments/README.md) | Earlier ROS PID controllers, real-world launch files, TF helpers, camera/SLAM/navigation bring-up traces, and workspace history | When reviewing how the ROS side evolved |
| [`arduino_firmware`](arduino_firmware/README.md) | Older Arduino balance-controller variants and `rosserial` publisher experiments | When checking Arduino-to-ROS history |
| [`raw_documents`](raw_documents/README.md) | Preserved source material, including the project trial-and-error PDF and receiver-interference notes | Only when the original archive material is needed |

For the cleaned main code paths, start with [`../firmware`](../firmware), [`../ros_ws`](../ros_ws), and [`../README.md`](../README.md).
