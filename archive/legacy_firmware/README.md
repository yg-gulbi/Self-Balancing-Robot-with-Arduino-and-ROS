# Legacy Firmware

This folder preserves older or alternate Arduino firmware variants.

- [`legacy_balance_controller.ino`](legacy_balance_controller.ino): earlier or alternate balancing controller variant with legacy `rosserial` traces for `/imu` and `/odom` publisher setup and helper functions; some publish calls are commented in this archived sketch
- [`experimental_balance_controller_imu_ros.ino`](experimental_balance_controller_imu_ros.ino): experimental firmware with `rosserial` IMU publishing and controller work

These files are kept for project history and evidence, not as the main firmware entry point.
